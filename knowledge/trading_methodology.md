## 方法论 #1：开盘区间止损法
**来源**：Minervini + Druckenmiller 联合讨论（2026-05-12 AAOX复盘）
**适用**：日内交易，尤其是高波动/杠杆标的

1. 开盘前30分钟不入场（等待区间确立）
2. 09:30–10:00 定义开盘区间 [L, H]
3. 突破 H 时入场，止损设在 L（或 L-1%）
4. VIX 同步上升时，区间不可信，推迟到 10:30
5. 止损随新高上移（不固定在开盘低点）

**反例**：今日AAOX开盘低点$61.78是VIX飙升引发的假低点，不可作为止损基准

## 方法论 #2：论点失效止损法
**来源**：Burry 提出，Marks 背书（2026-05-12）
**适用**：所有持仓，尤其是有消息面催化的仓位

入场前必须回答：**"当什么事情发生时，我的论点失效？"**

- 消息型催化：催化剂被证伪（如季报数据逆转）
- 技术型：趋势结构破坏（更低高点 + 更低低点）
- 板块型：同板块龙头趋势反转

**禁止**：单纯以"价格跌了 X%"作为唯一止损触发

## 方法论 #3：杠杆产品反脆弱仓位法
**来源**：Taleb（2026-05-12）
**适用**：2x/3x 杠杆 ETF

公式：**小仓位 × 宽止损 = 反脆弱**

- 仓位：账户的 3-5%（不超过）
- 止损宽度：-10% 到 -15%（开盘区间以外）
- 核心：止损位置不再被机构精准猎杀

## 方法论 #4：超买次日洗盘识别
**来源**：Marks（2026-05-12 AAOI复盘）
**适用**：前一日大涨（>15%）的高Beta股次日操作

规律：主升浪中，高位洗盘 = 机构清洗浮筹
- 信号：次日开盘跳空低开 + VIX同步上升 + 板块龙头（LITE）跌
- 操作：**不在跳空开盘后追止损，等30分钟后重新评估**
- 进阶：前一日已获利的，当天开盘前先减半仓锁利

## 方法论 #5：止损位置选择优先级
**综合讨论结论（2026-05-12）**

优先级从高到低：
1. **论点失效价格**（最优，不可被猎杀）
2. **开盘区间低点**（次优，基于当日结构）
3. **技术支撑位 -1%**（可接受，避开整数位）
4. **VWAP - X%**（备选，VWAP是机构成本参考）
5. ❌ **买入价 - X%**（最差，纯数学，易被猎杀）
6. ❌ **整数价位**（最差，$65/$70 是机构首选猎杀目标）

## 方法论 #6：连续 N 次信号确认入场法（agent team 审校版）
**来源**：用户实盘（2026-05-12）+ agent team 审查修正
**适用**：日内短线，高波动/杠杆标的

### 前置条件（全部满足才开始看信号）
1. **Stage 2 确认**：MA50 > MA150 > MA200，全部向上排列（Minervini）
2. **今日有催化剂**：新评级/业绩/重大新闻，不能纯靠技术信号（Burry）
3. **盘前 VIX 基准**：
   - VIX < 18 → N = 3次
   - VIX 18–20 → N = 5次（提高门槛）
   - VIX > 20 → 当日暂停2x杠杆入场（Druckenmiller）
4. **散户情绪**：Reddit/StockTwits 看多比例 < 70%（Marks）

### 信号条件（每2分钟，需连续 N 次全部满足）
- RS > +1.5%（vs SPY）
- MACD 柱状图 > 0
- RSI3 ≥ 50（不接受低于50，说明动能不足）
- R:R ≥ 1.5
- VIX 方向 →平 或 ↓降（盘中实时）

### 仓位 & 止损联动（Taleb + Minervini）
```
最大单笔亏损 = 账户总值 × 1%
结构止损    = 开盘区间低点（09:30-10:00）或底层标的关键支撑
仓位        = 最大单笔亏损 ÷ (入场价 - 结构止损价)
```
**止损和仓位必须联动设计，不可分开。**

### 原始版本 vs 修正版本
| 项目 | 原始（用户） | 修正（agent team）|
|---|---|---|
| 前置Stage分析 | 无 | Stage 2 必须确认 |
| 催化剂要求 | 无 | 今日需有具体论点 |
| VIX规则 | 盘中跟踪方向 | 盘前确定基准等级 |
| RSI3范围 | 35-75 | ≥50 |
| 散户情绪 | 无 | <70% |
| 止损仓位 | 分开设计 | 联动：先定亏损额→反推仓位 |

## 方法论 #8：大师综合止损框架（agent team 提炼版）
**来源**：10位大师研究 + agent team 讨论（2026-05-13）

### 止损触发优先级（从高到低）
1. **论点失效止损**（Burry+Livermore）：底层标的跌破关键支撑 AND 板块龙头同步转弱
2. **ATR结构止损**（Seykota）：入场价 - N×ATR（底层标的）；日内N=1，中线N=2
3. **箱体底部止损**（Darvas）：近5-10日整理低点下方1%
4. **均线止损**（Weinstein）：日线收盘跌破MA50（中线持仓）
5. ❌ **固定百分比**（O'Neil 7-8%仅适用普通股中线，不适用杠杆ETF日内）

### ATR仓位计算（Van Tharp × Seykota）
```
1. 确定可承受单笔亏损 = 账户总值 × 1%
2. ATR止损距离 = 底层标的的1日ATR
3. 杠杆ETF止损距离 = ATR × 杠杆倍数
4. 仓位 = 可承受亏损 ÷ 杠杆ETF止损距离
```

### 买入增强条件（加入方法论#6）
- **量能验证**（Zanger）：N次信号中至少1次量比>1.5x
- **散户情绪**（Marks+Burry）：社群看多<70%
- **均线方向**（Weinstein+Minervini）：MA50向上，底层标的在MA50上方

### 关键洞见
> "所有成功的止损系统，本质是论点管理，不是价格管理。" —— Burry×Livermore
> "仓位大小决定生死，止损只是执行工具。" —— Van Tharp×Taleb

## 方法论 #9：止损均线+缓冲带框架（场景分类版）
**来源**：agent team 讨论（2026-05-13）

### 止损前置判断：均线距现价的距离
```
距离 < 5%   → 用均线止损（MA + ATR缓冲）         [稳定上升趋势]
距离 5-15%  → 用结构止损（近期低点 + ATR缓冲）    [一般情况]
距离 > 15%  → 均线失效，用VWAP或论点失效止损      [急涨股，如AAOI]
```

### 均线止损计算（适用时）
```
缓冲带 = max(MA × 1%, 0.5 × ATR14)
止损线 = 参考均线 - 缓冲带
触发   = 收盘价 < 止损线 AND 成交量 > 20日均量
时间   = 允许1-2根K线跌破后确认（防假破）
```

### 均线选择
- 日内/短线：MA20
- 中线持仓：MA50
- 长线：MA150/30周线（Weinstein）

## 方法论 #10：反拥挤原则与有边界的噪声系统
**来源**：agent team 讨论（2026-05-13）
**核心洞见**：任何足够流行的策略，其参数本身成为猎杀目标

### 原则
> "策略的每个参数：原则确定，数值不固定。原则给边界，变化给不可预测性。"

### 入场噪声（边界：≤1个轮询周期）
- 第N次信号触发后不立即买，随机等待1-3分钟
- 或等首次小回调（0.3-0.5%）后以更低价格入场
- 避免与算法同时触发，降低追高概率

### 止损噪声（边界：基础止损的±1.5%）
- 实际止损 = 基础止损 × rand(98.5%, 99.7%)
- 绝不用整数价位（$65、$70是机构猎杀优先区）
- 止损位置以"非整数+非均线整数价"为原则

### 止损触发条件（反假破）
- 价格破位 AND 板块参照股同步弱 → 立即止损
- 只有价格破位，板块未跟跌 → 等1根K线确认收盘
- 边界：确认时间不超过1根轮询周期

### 维度多样化（最根本的反拥挤）
- 论点失效止损 > 价格止损（前者别人看不到，天然反拥挤）
- 用板块联动作为额外确认，不只盯单一股票价格

### 期望值边界公式（Taleb）
```
噪声最大边界 = 使策略期望值E仍然 > 0 的最大参数变化量
实用近似：
  止损噪声 ≤ 1.5%
  入场时间噪声 ≤ 1个轮询周期
  仓位噪声 ≤ 0.5个标准单位
```

## 方法论 #11：三档分批建仓体系
**来源**：用户+agent team 讨论（2026-05-13 盘后复盘）
**核心问题**：止损精细但建仓粗糙 = 非对称风险，需要建立与场景匹配的分批入场规则

### 核心原则
> "只向赢利方向加仓，永不摊平亏损。T1 亏损时不执行 T2。"

### 三档定义

| 档位 | 仓位比例 | 触发条件 | 止损调整 |
|------|---------|---------|---------|
| **T1 探测仓** | 1/3 | 6/6 入场信号满足 | 原始止损（结构低点-ATR×0.3） |
| **T2 确认仓** | 1/3 | T1入场后5分钟，价格仍在入场价上方+方向不变 | 止损上移至T1入场价（保本） |
| **T3 突破仓** | 1/3 | 价格突破关键压力位（VWAP/日高等），量比>1.2x | 止损上移至T2入场价 |

**失效规则**：T1 触发止损 → 立即清仓，取消 T2/T3 计划

### 全局禁止加仓条件（任一触发→不加仓）
```
量比>1.5x + 价格<T1成本×0.99  → 放量下跌（出货信号）
QQQ 5分钟跌幅 < -0.5%          → 大盘急跌
RSI3 < 35                      → 动能崩溃
价格 < VWAP × 0.97             → 深度跌破VWAP
```

### T2 两条路径（核心升级）

**路径A 强势加仓**（追涨型）
- 价格 > T1成本 × 1.005（至少高0.5%）
- 量比 > 0.85x（量正常，不是缩量拉）
- RSI3 < 78（未超买）
- QQQ 5分钟 > -0.2%

**路径B 回踩加仓**（洗盘型，往往更好的入场）
- 从T1回落 **0.3-2.5%**（不能回落太多）
- 量比 < 0.72x（缩量=无人抛售=洗盘）
- 价格 ≥ VWAP × 0.995（守住VWAP）
- RSI3 > 38（未崩溃）
- QQQ 5分钟 > -0.3%
- MACD柱 ≥ 0（趋势未逆转）

**关键原则**：量放大的下跌=出货，量缩小的下跌=洗盘，两者策略完全相反

### T3 突破加仓条件
- 价格接近或达到目标1（≥目标1×0.98）
- 量比 > 1.25x（放量确认）
- RSI3 < 82（未严重超买）

### 各场景分批策略

**A 强势突破**
- T1: 放量突破确认（量>1.3x）
- T2: 回踩不跌破突破位，5分钟后
- T3: 涨幅达到 ATR×0.5 后继续放量

**B VWAP回踩**（今日MRVL）
- T1: VWAP±1%区间内，3连阳+量缩
- T2: 5分钟后价格守住VWAP上方
- T3: 突破当日阻力位（如今日$180）

**D 冲高回落**（今日AAOI最佳操作）
- T1: 日低区域出现下影+3连阳（$192-198区间）
- T2: 量缩后5分钟仍在T1上方
- T3: 回收VWAP后（$202+）放量确认

**H 独立领涨**
- T1: 回踩VWAP，确认支撑
- T2: 5分钟守住，RS继续强于板块
- T3: 突破日高

### 仓位计算联动（与方法论#6/#8整合）
```
总风险预算 = 账户 × 1%
T1 止损距离 = 入场价 - 结构止损
T1 仓位 = 总风险预算 / T1止损距离
T2 止损 = T1入场价（风险归零）
T3 止损 = T2入场价（锁住T1利润）
```

### 今日教训（MRVL）
用户以$176.77全仓入场 → 应改为：
- T1 $176.77（1/3仓），止损$174.4
- T2 $177.5（+5分钟若守住），止损移至$176.77
- T3 $180突破（若量放大），止损移至$177.5

## 方法论 #12：催化剂强度分级与场景预测
**来源**：盘前新闻分析优化（2026-05-13）
**核心洞见**：催化剂强度决定高开后的走势方向，而非gap幅度本身

### 催化剂强度分级（0-3）

| 强度 | 类型 | 典型案例 | 高开后走势预期 |
|------|------|---------|-------------|
| ★★★ (3) | 战略入股/产品突破 | AMD入股MRVL、800G需求落地 | 可持续上涨，回踩VWAP是入场机会 |
| ★★☆ (2) | 财报超预期/重要合同 | EPS beat、大客户合同 | 中等，可能短暂sell the news后继续 |
| ★☆☆ (1) | 分析师调高/板块带动 | 目标价上调、板块涨带动 | 易sell the news，等回踩 |
| ☆☆☆ (0) | 负面消息 | 下调评级、指引低于预期 | 弱势整理或下跌 |

### 高开场景修正规则（原方法论D被修正）
- **高开>8% + 催化剂★★★** → 场景B（回踩VWAP入场），而非D（等回落）
- **高开>8% + 催化剂★☆☆** → 场景D（确实等冲高回落）
- 今日AAOI：分析师调高（★☆☆）→ D正确
- 今日MRVL：AMD战略入股（★★★）→ 应判断B，实际也是VWAP回踩后继续涨

### 同一消息对买方/卖方的不同影响
> AMD披露持有MRVL：MRVL+8.2%，AMD-0.6%
> 原则：战略入股消息看**被投资方**，投资方本身往往"买消息卖事实"
